สร้างไฟล์ index.html เป็นไฟล์ HTML เป็นระบบช่วยเลือกร้านอาหารสำหรับทีม 8–12 คน จากข้อมูลที่ scrape มา พร้อม scoring, scenario recommendation และ AI Ask Box แบบอิง dataset

Web Component
1. มีปุ่ม Scrape ข้อมูลจาก APIFY โดยเลือกทำการ Scrape ร้านอาหารในย่าน อารีย์ ทองหล่อ พร้อมพงศ์ สยาม อโศก อย่างละ 20 ร้าน โดยดึงข้อมูลมาอย่างน้อย 20 ร้าน เก็บข้อมูลไว้ใน googlesheet
    - ฟิลด์ที่ทำการ scrape ได้แก่ ชื่อร้าน พื้นที่ ประเภทอาหาร คะแนนรีวิว จำนวนรีวิว ช่วงราคา จำนวนโต๊ะ ที่ตั้ง หมายเหตุการเดินทาง เวลาเปิดปิด เหมาะกับกลุ่มหรือไม่ ลิงก์แหล่งข้อมูล หมายเหตุเพิ่มเติม
    - เก็บข้อมูลไว้ใน Google Sheet ชื่อไฟล์ Restaurant dataset
2. แสดงจำนวนข้อมูลร้านที่มีทั้งหมดในย่านต่างๆ ที่กำหนด
3. มีส่วนสำหรับตั้งค่า (Settings) เพื่อให้ผู้ใช้สามารถระบุข้อมูลตามนี้:
    - ปลอดภัยสูงสุด
    - ประหยัด (ราคาต่อหัวน้อยที่สุด)
    - รีบกิน (บริการเร็ว ใช้เวลาน้อย)
    - คุยงานได้ (เงียบ บรรยากาศเหมาะกับการประชุม)
    - ระบุจำนวนผู้ร่วมเดินทาง 8–12 คน (โต๊ะใหญ่ รองรับกลุ่ม)
4. ระบบจะช่วยวิเคราะห์และแสดงร้านอาหารแนะนำ โดยอิงจากข้อมูลที่scrape มา พร้อมแสดงเหตุผล (Justification) และค่าคะแนน (Score) สำหรับแต่ละร้าน โดยมีการให้คะแนนจากปัจจัยดังต่อไปนี้้
    Rating 25% / Group 20% / Price 15% / Travel 15% / Data 15% / Unique 10%
5. มีช่องสำหรับพิมพ์ข้อความสอบถามเพิ่มเติม (AI Ask Box) ที่เชื่อมต่อ API LLM โดยข้อความที่ถามจะอ้างอิงจากข้อมูลร้านที่แสดงอยู่ และจะดึงข้อมูลที่เกี่ยวข้องจาก googlesheet (ร้านอาหาร, สถานที่, เวลาเปิดปิด, ที่จอดรถ, บรรยากาศ, ราคา) เพื่อมาประกอบการตอบ
6. ลิสต์ร้านอาหารพร้อมฟิลเตอร์ต่างๆ ดังนี้
    - ประเภทของร้านอาหาร (ร้านอาหารญี่ปุ่น / เกาหลี / จีน / ไทย / อิตาเลียน / ปิ้งย่าง /อื่นๆ )
    - Scenario (เหมาะคุยงาน / กินเร็ว / ราคาถูก)
    - ย่านอาหาร ( อารีย์ ทองหล่อ พร้อมพงศ์ สยาม อโศก)
    - ราคาต่อหัว
    - จำนวนโต๊ะ
7. หากข้อมูลที่มีไม่พอ ควรมีปุ่มเพื่อให้สามารถ Scrape ข้อมูลเพิ่มเติมจาก APIFY ในย่านเดิมได้
8. สามารถดูตัวอย่างได้ที่ https://ai-food-assistant-one.vercel.app/
9. สไตล์เว็บแบบ Dark Mode สวยงาม มี UI/UX ที่ดี
